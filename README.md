# AISYNC-DEMO-MVP

AISYNC-DEMO-MVP is the minimum functional demo of AISync.

It focuses on:

- single-user AISync cell;
- internal multi-teams;
- workspace per team;
- Save Version / Checkpoint;
- basic Audit Log;
- basic Documentation Mode;
- future-ready structure for Organizational Elasticity.

Core loop:

Create → Work → Save → Trace → Consult → Resume

See:

- `docs/MVP_SCOPE.md`

# AISync Demo

Local demo for AISync (Austin) showcasing the UI and workflow experience.

## Run locally

```bash
npm install
npm run dev
```

## Notes

This demo runs locally with Vite and is intended for UI and workflow demonstration.
